# Edge 🎙️
**Smart ESP32-based digital assistant and social discovery device.**

Edge — це компактний портативний пристрій на базі мікроконтролера ESP32, який поєднує в собі функції цифрової шпаргалки, диктофона з транскрипцією та інтелектуального ШІ-асистента. 

## 🌟 Основні можливості
* **Голосове введення (I2S):** Запис аудіо через цифровий мікрофон INMP441.
* **Транскрипція та ШІ (Wi-Fi):** Відправка аудіо на проксі-сервер для розпізнавання (Whisper) та генерації відповідей (GPT-4o-mini).
* **Читалка файлів (SD-карта):** Зчитування згенерованих відповідей та текстових файлів з SD-карти з підтримкою пагінації.
* **OLED Інтерфейс:** Зручне меню з іконками, відображенням рівня заряду батареї та статусу Wi-Fi.
* **PRO Режим:** Система підписок з перевіркою статусу через MAC-адресу пристрою (збільшений час запису, розширені відповіді).
* **OTA Оновлення:** Підтримка бездротового оновлення прошивки з можливістю відкату (Rollback) через меню налаштувань.
* **Web-конфігуратор:** Вбудована точка доступу (`ESP32-Control-Panel`) для підключення до домашньої мережі.

## 🛠 Апаратна частина (Pinout)
Пристрій розроблено для роботи з ESP32. Нижче наведено схему підключення компонентів:

| Компонент | Пін ESP32 | Опис |
| :--- | :--- | :--- |
| **OLED (I2C)** | `21` (SDA), `20` (SCL) | Екран SSD1306 (128x64) |
| **Мікрофон (INMP441)** | `2` (SD), `9` (WS), `10` (SCK)| Цифровий I2S мікрофон |
| **SD-карта (SPI)** | `3` (CS), `4` (MISO), `5` (MOSI), `6` (SCK) | Зберігання відповідей та аудіо |
| **Кнопка Вгору** | `8` | Навігація / Запуск Web-сервера (Hold) |
| **Кнопка OK** | `7` | Вибір / Запис аудіо / Меню (Hold) |
| **Кнопка Вниз / Батарея**| `1` (HYBRID) | Гібридний пін: кнопка + АЦП акумулятора |

## 📚 Залежності та бібліотеки
Для компіляції проєкту в Arduino IDE або PlatformIO необхідні наступні бібліотеки:
* `U8g2` (для роботи з OLED дисплеєм)
* `ArduinoJson` (для парсингу API-відповідей)
* `EncButton` (для обробки кліків та утримання кнопок)
* Стандартні бібліотеки ESP32: `WiFi`, `WiFiClientSecure`, `SD`, `SPI`, `Preferences`, `Update`.

## 🚀 Встановлення та прошивка
1. Клонуйте цей репозиторій.
2. Відкрийте проєкт в Arduino IDE.
3. Оберіть вашу плату ESP32 (наприклад, ESP32S3 Dev Module).
4. **Важливо:** У налаштуваннях плати (Tools -> Partition Scheme) обов'язково оберіть схему з підтримкою OTA (наприклад, **"8MB with spiffs (3MB APP/1.5MB SPIFFS)"** або **"Minimal SPIFFS (Large APPS with OTA)"**). Без цього відкат та оновлення по Wi-Fi не працюватимуть.
5. Скомпілюйте та завантажте код.

## 🌐 Робота з сервером (Proxy)
Для безпеки API-ключів (OpenAI) пристрій не робить прямих запитів до зовнішніх сервісів. Замість цього він спілкується з Cloudflare Worker (`esp32-ai-proxy...`), який обробляє транскрипцію аудіо, перевіряє PRO-статус за MAC-адресою та передає запити до GPT.
